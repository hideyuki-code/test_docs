# プロトタイプおよび要件定義・設計書リポジトリ

このリポジトリは、現在進行中のプロジェクトのプロトタイプを管理するとともに、要件定義および設計書を作成・共有するための場所です。

## 現在の目的

- プロジェクトの初期検証のためのプロトタイプ実装を管理する
- 要件定義書や設計書などのドキュメントを体系的に整理する
- 関係者間での情報共有と合意形成を支援する

## リポジトリ構成（予定）

- `docs/` : 要件定義書・設計書などのドキュメント
- `prototype/` : プロトタイプ実装
- `rules.md` : 運営ルールや作業指針

※構成は必要に応じて更新されます。

## 今後の進め方

- プロトタイプの更新やドキュメント追加の際はプルリクエストを作成してレビューを行う
- ルールや手順の変更があれば `rules.md` を更新し共有する
