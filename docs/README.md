# ドキュメントトップ

このサイトは Docsify により Markdown をブラウザでレンダリングしています。

- 左のサイドバーから各仕様書へ移動できます
- プロトタイプは「プロトタイプ」セクションからアクセスしてください

## 構成

- docs/ ... 仕様書（Markdown）
- docs/prototypes/ ... 単一HTMLのプロトタイプ
- schema/ ... ER図などのスキーマ関連

## 運用メモ

- 変更を `main` に push すると自動デプロイ（Vercel / Hobby）
- PR を作るとプレビューURLが発行されます
