## 3. 機能

### **3-1.** システム共通

| 機能名 | 概要 | 詳細 | 対象 |
| --- | --- | --- | --- |
| **閲覧** | 一覧表示 | 姓
名
社員番号
メールアドレス
部署
権限
招待ステータス | 全ユーザー |
| **詳細閲覧** | 行をクリックで詳細ページに遷移 | - | adminユーザー |
| 編集閲覧 | 編集ボタンをクリックで編集ページに遷移 | - | adminユーザー |
| 登録（ユーザ招待） | ユーザ招待ボタンをクリックで新規ユーザ招待モーダルが開く | -  | adminユーザー |
| **削除** | レコードを削除 | 確認モーダルを経て削除実行 | adminユーザー |
| **フィルター** | 条件で絞り込み表示 | URLパラメータに反映、選択中ラベルを表示 | 全ユーザー |
| **ページネーション** | 表示件数・ページ移動 | localStorageに記憶（デフォルト50件） | 全ユーザー |
| **カラムソート** | 任意カラムで昇順／降順 | クリック操作で切り替え | 全ユーザー |

### **3-2. admin機能**

| 機能名 | 概要 | 詳細 | 対象 |
| --- | --- | --- | --- |
| 招待（ユーザー新規作成） | ユーザーの新規作成&招待 | モーダルでformを入力し「招待を送る」ボタンをクリックすると、以下が実行される：
1. 入力したアドレス宛に招待メールを送信
2. 「招待ステータス」= 「招待中」のレコードを作成

招待メールには以下を含む：
・ビクロスのログインURL
・アカウントが発行された旨 | adminユーザー |

**モーダル入力項目:**

| (* = 必須)項目名 | 型 | placeholder | 制約 | 備考 |
| --- | --- | --- | --- | --- |
| *姓 | string |  | - | - |
| *名 | string |  | - | - |
| *セイ | string |  | - | - |
| *メイ | string |  | - | - |
| *社員番号 | string |  | ユニーク、4桁 | 例: 0012, 0294, 0314 |
| *address | string |  | ユニーク | メールアドレス |
| *権限 | ENUM |  | [管理者、編集者、閲覧者](admin, writer, reader) | デフォルト値 = writer (編集者) |
